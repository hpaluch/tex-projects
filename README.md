Here are my Tex Projects archive

##Build Install

Requires csTex and/or csLatex. On (L)Ubuntu 14 use this command to install:

	sudo apt-get install texlive-lang-czechslovak

To compile PDF file from TEX (and MF) use:

	cd <SUBPROJECT_NAME>
	make
